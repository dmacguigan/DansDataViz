Today we’re going to explore some sports data. While watching my favorite hockey team this year (the Philadeplphia Flyers), I’ve felt like I barely know the players. There was a lot of turnover in the roster in the past couple of years.

This got me thinking. Which year did the Flyers roster experience the most turnover?

To answer this question, we will turn to the NHL’s own stats database. Annoyingly, we can only download 50 records at a time using their nice web browser interface. So instead, we’ll need to use an Application Programming Interface, or API.

Fortunately, some folks have put together nice documentation here and here for the NHL API.

To start, we’ll use the requests and json Python libraries. Make sure you have them installed.

Let’s do a toy example where we pull the current NHL standings. I’m pulling heavily from this tutorial.
